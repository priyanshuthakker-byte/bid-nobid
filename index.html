How Rules Work:
1. DO NOT BID — if any keyword found in tender brief → immediate NO-BID (cannot be overridden)
2. CONDITIONAL — if keyword found → CONDITIONAL (needs manual check)
3. PREFERRED SECTORS → BID (only if no DO NOT BID rule fired)
Compound Rules: Use pipe | to separate MUST keywords from EXCLUDE keywords. E.g. gis|supply,procurement means: tender must contain "gis" but NOT "supply" or "procurement"
